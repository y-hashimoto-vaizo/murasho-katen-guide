はじめに

このたびは、新事業進出補助金 第4回公募に再チャレンジいただきありがとうございます。

補助金の審査では、「加点要件」という項目を取得しているかどうかで採択率が大きく変わります。

本書では、川上社長に取得いただきたい 3つの加点要件 について、それぞれの手順を丁寧にご案内します。

いずれも完全無料で、すべてパソコンからの登録作業のみで完了します。認定機関の審査待ちなどは発生しませんので、ご安心ください。

本手続きは貴社にて実施いただきます。公開までの日数を考慮すると余裕を持った着手が必要ですので、4月中にすべてのお手続きを完了いただけますようお願い申し上げます。

パートナーシップ構築宣言

取引先との共存共栄を宣言する、国の制度です。3つのうち最も簡単なので、まずここから取り組みます。

所要時間
登録 約1時間
公開までの日数
約10日
費用
無料
公式サイト
biz-partnership.jp

手順登録の流れ

  1. 公式サイトで制度の内容を確認 biz-partnership.jp/outline.html にアクセスし、宣言の内容をご一読ください。
  2. 宣言文のひな形（Word）をダウンロード biz-partnership.jp/register.html の「記載要領・ひな形」から、最新の2026年1月版をダウンロードします。 ひな形には「記載要領」も併せて掲載されています。必ず記載要領に沿ってご記入ください。
  3. 宣言文の内容をご確認・ご決裁 代表者名義で公開される宣言文ですので、川上社長に内容をご確認いただきます。
  4. PDFに変換して保存 Wordファイルを PDF 形式で保存します。
  5. ポータルサイトから登録 登録フォームに会社情報を入力し、PDFをアップロードして「登録する」をクリックします。
  6. 受領メールを確認 担当者のメールアドレスに「『パートナーシップ構築宣言』を受領しました」というメールが届きます。
  7. 約10日後、公開を確認 ポータルサイトの「登録企業リスト」に会社名が表示されます。公開画面のスクリーンショットを保存してください。 スクリーンショットは Windows キー ＋ Shift ＋ S で画面の一部を切り取って保存できます。
⚠ 繁忙期は公開まで少し日数がかかります 補助金申請が集中する時期は、公開まで10日＋数日かかることがあります。4月中に登録を完了してください。
✅ 補助金申請時に必要な証拠書類 ・ポータル掲載のURL
・公開画面のスクリーンショット

22つ目の加点要件

成長加速マッチングサービス

中小企業庁が運営している、事業者と支援機関をつなぐマッチングサービスに登録します。

所要時間
約30分
GビズID
必須です
費用
無料
公式サイト
seichokasoku.smrj.go.jp

手順登録の流れ

  1. 公式サイトにアクセス seichokasoku.smrj.go.jp をパソコンで開きます。
  2. 右上の「新規登録・ログイン」ボタンを押す 画面の右上にあるオレンジ色のボタンです。
  3. 「事業者」を選ぶ 「事業者」と「支援者」の選択画面が出るので、事業者を選んでください。
  4. GビズIDプライムでログイン 貴社のGビズIDでログインします。
  5. 利用規約に同意 チェックを入れて次へ進みます。
  6. 会社情報・担当者情報を入力 画面の指示に従って入力していきます。
  7. 「挑戦課題」を1件投稿 新しいEC事業に関する課題を1つ投稿します。以下のような内容で貴社にて文章をご作成ください。 参考例：「全国向け建材EC事業立ち上げに伴う、AIコンシェルジュ機能の開発パートナー募集」／「建築資材ECにおける物流アライアンス先の募集」など。
  8. 数営業日後、ステータスを確認 投稿した課題のステータスが「掲載中」になったら、画面のスクリーンショットを保存してください。
📝 補助金の要件について 応募締切日の時点で「掲載中」の課題が1件以上あれば加点要件を満たします。4月中に登録を済ませてください。
✅ 補助金申請時に必要な証拠書類 ・会員登録完了画面のスクリーンショット
・「掲載中」になっている挑戦課題画面のスクリーンショット

33つ目の加点要件

両立支援のひろば

仕事と家庭の両立を支援する計画を作り、厚生労働省のサイトで公表します。少し手順が多いですが、1つずつ進めれば約1.5時間で完了します。

所要時間
約1.5時間
公開までの日数
4営業日
費用
無料
公式サイト
ryouritsu.mhlw.go.jp
📝 村商東北様（従業員20名）について 法律上、従業員100人以下の企業は労働局への届出は「努力義務」ですが、補助金の加点要件では届出まで必要とされていますので、本書の手順では届出までご案内します。

手順 1目標を1〜2個決める

法律で「仕事と子育ての両立を支援する目標」を1つ以上決める必要があります。村商東北様で無理なく取り組めそうな候補をご用意しました。

この中から1〜2個お選びください。未達成でも罰則はございません。

  • 候補A男性従業員の育児休業取得率を、計画期間中に1名以上達成する
  • 候補B年次有給休暇の年間平均取得日数を、従業員1人あたり○日以上にする
  • 候補C所定外労働時間の削減のため、毎週○曜日をノー残業デーにする
  • 候補D育児をする従業員のための短時間勤務制度を導入する
  • 候補E子の看護休暇の取得を社内に周知する

手順 2両立支援のひろばに新規登録（約15分）

  1. サイトへアクセス ryouritsu.mhlw.go.jp を開きます。
  2. 「企業の方」メニュー → 一般事業主行動計画公表サイト へ トップページから進みます。
  3. 「新規登録」ボタンをクリック メールアドレス、会社名、所在地、業種、従業員数、担当者情報を入力します。
  4. 登録完了メールで初回ログイン 届いたメールのリンクからログインします。

手順 3行動計画を作成する（約1時間）

  1. マイページから「行動計画を作成する」を選ぶ
  2. 「全分野を回答する」を選択
  3. 計画期間を入力 推奨：2026年5月1日〜2028年4月30日（2年間）
  4. 手順1で決めた目標を選ぶ
  5. 目標達成のための取組内容を選択 サイトに用意された候補から該当するものをクリックするだけで入力できます。
  6. 公表方法・周知方法を選ぶ 「両立支援のひろばへの掲載」＋「社内掲示」を選びます。
  7. 行動計画書をPDFで保存・印刷 回答が終わると、厚労省の様式に沿った計画書が自動で作られます。

手順 4社内で従業員に周知（約30分）

法律で従業員への周知が必要です。次のいずれか1つで大丈夫です。

  • 方法A社内の見やすい場所（休憩室、社員通用口など）に掲示する
  • 方法B社内メールや社内SNSで全従業員に配信する
  • 方法C社内イントラネットに掲載する
⚠ 必ず記録を残してください 掲示した場所の写真、または配信メールのスクリーンショットを保存してください。補助金申請の証拠になります。

手順 5両立支援のひろばで公表する（約10分）

  1. マイページから「行動計画を公表する」を選ぶ
  2. 送信する 土日祝を除く約4営業日以内に公開されます。
  3. 公開後、URLとスクリーンショットを保存

手順 6宮城労働局に届出する（約20分）

次のいずれかの方法で届け出ます。弊社としてはオンライン（e-Gov）をおすすめします。

方法A：オンライン届出（推奨）
e-Gov電子申請 で「次世代育成支援対策推進法」を検索し、「一般事業主行動計画策定・変更届」から提出します。
方法B：書面で郵送
様式第1号を印刷・記入し、下記宛に郵送。
〒983-8585 仙台市宮城野区鉄砲町1番地 仙台第4合同庁舎8階
宮城労働局 雇用環境・均等室
✅ 補助金申請時に必要な証拠書類 ・両立支援のひろば 公開URL
・公開画面のスクリーンショット
・宮城労働局への届出控え（e-Govの完了通知 or 郵送の受理印写し）
